Local-first Web MVP

읽고, 찾고, 미리 보고, 승인 후 저장합니다

이 웹 셸은 브라우저에서 파일 요약, 문서 검색, 세션 기록, 승인 기반 저장 흐름을 한 번에 확인하는 데모용 MVP입니다. 기본값은 읽기 중심이며, 위험한 쓰기 작업은 항상 별도 승인 카드에서 멈춥니다.

현재 기본값

앱 __APP_NAME__
프로바이더 __DEFAULT_PROVIDER__
모델 __DEFAULT_MODEL_LABEL__
현재 세션 __DEFAULT_SESSION_ID__
웹 검색 __DEFAULT_WEB_SEARCH_PERMISSION_LABEL__

세션과 대화 기록

__DEFAULT_SESSION_ID__ 아직 메시지가 없습니다.

최근 세션

검토 후보 현재 후보는 검토 수락만 기록할 수 있습니다. 아직 적용, 편집, 거절은 열지 않았습니다.
검토 메모 적용 후보 현재 반복 교정 묶음은 aggregate 단위 경계만 보여주며, 아직 시작할 수 없습니다.

요청 보내기

파일 요약, 문서 검색, 일반 채팅을 한 화면에서 전환할 수 있습니다.

모드

브라우저 파일 선택창으로 직접 고른 파일은 로컬 서버로만 전달되며, 절대경로 대신 파일명 기준으로 처리됩니다.

브라우저 폴더 선택창으로 직접 고른 파일은 로컬 서버로만 전달되며, search_root 대신 선택 파일 목록 기준으로 검색합니다.

고급 설정

외부 웹 검색은 읽기 전용으로 연결되어 있고, 검색 결과 요약과 링크/스니펫을 세션별 로컬 기록으로 저장합니다. 지금은 차단/승인 필요/허용 상태를 세션별로 나눠 두고 있습니다.
