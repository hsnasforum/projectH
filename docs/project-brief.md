# Project Brief

## Product One-Liner

A local-first AI assistant that helps users read, summarize, search, and save information from personal files while requiring explicit approval for risky actions.

## Current Goal

Ship a small, safe MVP for personal document work.

## MVP Must Include

- chat interaction
- local file reading
- document summarization
- document search
- session persistence
- approval-based write actions
- task logs

## MVP Must Not Include By Default

- wake-word voice assistant
- mobile app
- messenger integrations
- browser login automation
- autonomous destructive actions
- large-scale independent pretraining

## Product Constraints

- local-first by default
- commercial and branding clarity
- replaceable model/runtime layer
- safe, reversible operations
