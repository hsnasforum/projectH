# Recycle Documents

This folder keeps **retained but non-source-of-truth** documents.

Use it for:
- historical proposals
- working drafts that informed the current product direction
- notes kept for reference but not used as the current implementation contract

## Structure

- `drafts/`
  - working drafts and direction notes
- `legacy-proposals/`
  - earlier proposal documents kept for reference
- `legacy-reference/`
  - older architecture, API, schema, and planning references no longer used as the current contract

## Source of Truth

Current source-of-truth product and engineering docs live in the root `docs/` directory:
- `project-brief.md`
- `PRODUCT_PROPOSAL.md`
- `PRODUCT_SPEC.md`
- `ARCHITECTURE.md`
- `ACCEPTANCE_CRITERIA.md`
- `MILESTONES.md`
- `TASK_BACKLOG.md`
- `NEXT_STEPS.md`
