# 웹 검색 UI 설계

**작성일**: 2026-04-02
**목표**: `/검색` 슬래시 명령어로 웹 검색을 명시적으로 트리거

## 입력 방식
- 입력창에 `/` 입력 시 자동완성 드롭다운 표시
- `/검색 나루토 줄거리` → 웹 검색 실행
- `/요약`, `/메모` 등 향후 확장 가능

## 결과 표시
- 채팅 안에 인라인 (현재 방식 유지)
- 백엔드 변경 없음

- `InputBar.tsx` — `/` 감지 + 드롭다운 UI
- `useChat.ts` — `/검색 ` prefix 파싱, 웹 검색 파라미터 전달
